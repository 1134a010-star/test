# Echoes of Antiquity | 歐洲古典藝術史

一個沉浸式的、具有視差滾動效果的歐洲古典藝術史介紹網頁。本項目採用抽象且高質感的設計風格，展現古希臘羅馬、文藝復興與巴洛克時期的藝術精神。

## 專案功能
- **沉浸式視差滾動 (Parallax Scrolling)**：利用 CSS 3D 轉換實現多層次的深淺透視感。
- **動態揭示動畫 (Reveal Animations)**：使用 `IntersectionObserver` 在滾動時觸發各章節內容的優雅呈現。
- **抽象美學設計**：結合幾何圖形與高對比色調（黑、金、米色），打造現代且尊貴的視覺體驗。

## 技術棧
- **HTML5**: 語義化結構。
- **CSS3**: 視差效果、CSS 變數、Flexbox、Grid。
- **JavaScript (ES6+)**: 內容監聽與動畫邏輯。

## 如何運行
直接在瀏覽器中開啟 `index.html` 即可檢視。推薦搭配 [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) 以獲得最佳開發體驗。
